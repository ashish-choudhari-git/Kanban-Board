# Kanban Board

A simple Kanban Board application built using **HTML, CSS, and Vanilla JavaScript**.

## Features

* Create new tasks
* Delete tasks
* Drag and drop tasks between columns (todo,in progress, done)
* Task count updates automatically
* Data persists using Local Storage

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
